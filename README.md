# yii2-paypal-sdk
Yii2 basic + paypal component (paypal-sdk based) 

Install:
1. Clone or download 
2. Configure data base
3. php yii migrate/up --migrationPath=@vendor/budyaga/yii2-users/migrations
4. php yii migrate/up --migrationPath=migrations
